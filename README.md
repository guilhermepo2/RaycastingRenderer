# RaycastingRenderer

Ideally, the renderer should be cross-platform by default, but specifically it was developed on the Debian GNU/Linux Windows Subsystem for Linux (WSL).

Dependencies:
* SDL2

Install SDL2 by:

```
sudo apt-get install libsdl2-dev
```
